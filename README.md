# Suomen Palikkaharrastajat ry — Brand Guide

Brand assets for Suomen Palikkaharrastajat ry. The logo mark is a LEGO minifig
head rendered as brick art using the `brick_blockify.py` pipeline.

## Brand Colors

| Color | Hex |
|-------|-----|
| **LEGO Yellow** | `#F2CD37` |
| **LEGO Black** | `#05131D` |
| **LEGO White** | `#FFFFFF` |

Edit `colors.py` to change the palette; all outputs are regenerated by `make`.

## Logo Variants

### Square (profile picture / icon)

| Variant | Description |
|---------|-------------|
| Yellow | Classic minifig head, SKIN_TONES[0] |
| Colorful skin tones | Horizontal bands of all four skin tones |

Files: `logo/square/svg/`, `logo/square/png/` (800 px PNG + WebP; GIF for colorful)

### Horizontal (four heads with subtitle)

Four minifig heads side by side, each with a different LEGO skin tone.

| Variant | File stem |
|---------|-----------|
| Brick art only | `horizontal` |
| With association name subtitle | `horizontal-full` |

Files: `logo/horizontal/svg/`, `logo/horizontal/png/` (800 px PNG + WebP)

## Build

Requires Python 3 with **Pillow** and **cairosvg** (provided by `devenv.nix`).

```bash
make          # build everything (design sources → brick SVGs → PNG/WebP/GIF)
make designs  # populate design/ from colors.py
make logos    # brick SVGs only
make pngs     # PNG + WebP + GIF exports
make clean    # remove all generated files
```

## Pipeline

```
colors.py + generate_designs.py
     │ design/square.svg            → brick_blockify.py → logo/square/svg/square.svg
     │ design/horizontal.svg        → brick_blockify.py → logo/horizontal/svg/horizontal.svg
     │                                                             │
     │                                                             ▼
     │                                                      compose_logo.py
     │                                                             │
     │                                                             ▼
     │                                                   horizontal-full.svg
     │
     └ design/minifig-colorful.svg  → brick_blockify.py → logo/square/svg/minifig-colorful.svg
                                            │                        │
                                            ▼                        ▼
                                     animate_logo.py          export_raster.py
                                            │                        │
                                            ▼                        ▼
                                   minifig-colorful.gif    minifig-colorful.{png,webp}

All SVGs → export_raster.py → 800 px PNG + WebP
```

## Source Files

| File | Purpose |
|------|---------|
| `colors.py` | Brand colors, skin-tone palette, association name |
| `generate_designs.py` | Generates all design SVGs from `colors.py` |
| `brick_blockify.py` | Converts design SVG → brick-art SVG |
| `compose_logo.py` | Appends subtitle text to a horizontal brick SVG |
| `export_raster.py` | Exports SVG → PNG / WebP |
| `animate_logo.py` | Generates animated GIF from rotating skin-tone bands |

## Customisation

- **Colors**: edit `colors.py` and run `make`.
- **Skin tones**: edit `SKIN_TONES` in `colors.py`; affects all logo variants.
- **Brick size**: adjust `SQ_PX` / `HZ_PX` / `BLK_W` / `BLK_H` in the Makefile.
