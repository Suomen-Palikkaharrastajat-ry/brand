# Agent instructions

## Build commands

All build and Elm-related commands must be run inside the devenv shell:

```
devenv shell -- elm-pages build
devenv shell -- elm-pages dev
devenv shell -- elm make ...
devenv shell -- make site
devenv shell -- make assets
devenv shell -- make blay-compose
```

Do **not** use `npx`, bare `elm-pages`, or bare `elm` — they are not on PATH outside devenv.

## Project overview

Elm + elm-pages design guide site for Suomen Palikkaharrastajat ry.
Brand assets (SVGs, PNGs, favicons) and design tokens are generated by a Haskell pipeline.

Key directories:
- `src/Component/` — Elm UI component library
- `app/Route/` — elm-pages routes
- `src/Brand/` — Haskell brand pipeline and generated Elm tokens
